"mass:/RETROLauncher/Roms/Roms Atari 2600/Example.a26" - It will find it and list it.
"mass:/RETROLauncher/Roms/Roms Atari 2600/Example.bin" - It will find it and list it.
"mass:/RETROLauncher/Roms/Roms Atari 2600/Example.zip" - It will find it and list it.
